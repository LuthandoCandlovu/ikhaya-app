## рҹҡҖ Quick Start

### рҹ“Ӣ Prerequisites

Before you begin, ensure you have the following installed:

```
вң… Node.js 16+
вң… npm or yarn
вң… Expo CLI (global)
вң… iOS Simulator (macOS) or Android Studio Emulator
вң… A physical device with Expo Go app (optional)
```

<br/>

### вҡҷпёҸ Installation

**1. Clone the Repository**

```bash
git clone https://github.com/your-username/khaya-app.git
cd khaya-app
```

**2. Install Dependencies**

```bash
npm install
# or
yarn install
```

**3. Install Expo CLI Globally**

```bash
npm install -g expo-cli
```

**4. Configure Environment Variables**

```bash
cp .env.example .env
```

Open `.env` and fill in your credentials:

```env
# Firebase Configuration
FIREBASE_API_KEY=your_firebase_api_key
FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_STORAGE_BUCKET=your_project.appspot.com
FIREBASE_MESSAGING_SENDER_ID=your_sender_id
FIREBASE_APP_ID=your_app_id

# AI Services
OPENAI_API_KEY=your_openai_api_key
DIALOGFLOW_PROJECT_ID=your_dialogflow_project_id

# Backend
API_BASE_URL=http://localhost:3000
```

**5. Start the Development Server**

```bash
npx expo start
```

Then scan the QR code with **Expo Go** on your device, or press:
- `i` вҶ’ Open iOS Simulator
- `a` вҶ’ Open Android Emulator
- `w` вҶ’ Open in Web Browser
