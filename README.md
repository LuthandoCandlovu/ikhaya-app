## 🌍 What is Khaya?

<img align="right" width="380" src="https://media.giphy.com/media/LaVp0AyqR5bGsC5Cbm/giphy.gif"/>

**Khaya** *(isiZulu for "Home")* is the campus living upgrade every student deserves.

No paper forms. No email chains. No notice boards.

Open Khaya → Speak or type → Done. ✅

```bash
"Hey Khaya, broken light in room 14B"
  → Ticket created          ✅
  → Staff notified          🔔
  → You track it live       📲
```

> 💜 Built on **Ubuntu** — *"I am because we are"*

## 🚀 Quick Start

<div align="center">
<img src="https://media.giphy.com/media/LmNwrBhejkK9EFP504/giphy.gif" width="180"/>

<img src="https://readme-typing-svg.demolab.com?font=Fira+Code&size=15&duration=1800&pause=600&color=34D399&center=true&vCenter=true&width=480&lines=Clone+→+Install+→+Configure+→+🚀+Launch" alt="Steps"/>
</div>

<br/>

**① Clone the repo**
```bash
git clone https://github.com/your-username/khaya-app.git
cd khaya-app
```

**② Install dependencies**
```bash
npm install
npm install -g expo-cli
```

**③ Set up your environment**
```bash
cp .env.example .env
```
```env
# ── Firebase ──────────────────────────────────────────
FIREBASE_API_KEY=your_firebase_api_key
FIREBASE_AUTH_DOMAIN=yourproject.firebaseapp.com
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_STORAGE_BUCKET=yourproject.appspot.com
FIREBASE_MESSAGING_SENDER_ID=your_sender_id
FIREBASE_APP_ID=your_app_id

# ── AI ────────────────────────────────────────────────
OPENAI_API_KEY=your_openai_api_key
DIALOGFLOW_PROJECT_ID=your_dialogflow_id

# ── Backend ───────────────────────────────────────────
API_BASE_URL=http://localhost:3000
```

**④ Fire it up 🔥**
```bash
npx expo start

# i → iOS Simulator
# a → Android Emulator
# w → Web Browser
```
