## 🌊 What is Khaya?

</div>

<img align="right" src="https://raw.githubusercontent.com/platane/snk/output/github-contribution-grid-snake-dark.svg" width="45%"/>

**Khaya** *(isiZulu for "Home")* is an AI-powered mobile companion that transforms how university students interact with their campus residence.

No more paper forms. No more lost emails. No more missed announcements.

Just open Khaya, speak or type, and it's done.

```
"Hey Khaya, report a broken geyser in room 14B"
→ Maintenance request logged ✅
→ Staff notified instantly 🔔  
→ You get real-time updates 📲
```

> 💡 Built on the philosophy of **Ubuntu** — *"I am because we are"*

## 🚀 Quick Start

<img src="https://readme-typing-svg.demolab.com?font=Fira+Code&size=14&duration=1800&pause=600&color=34D399&center=true&vCenter=true&width=500&lines=Clone+→+Install+→+Configure+→+Launch+🚀" alt="Quick Start Steps"/>

</div>

**Step 1 — Clone**
```bash
git clone https://github.com/your-username/khaya-app.git && cd khaya-app
```

**Step 2 — Install**
```bash
npm install && npm install -g expo-cli
```

**Step 3 — Configure Environment**
```bash
cp .env.example .env
```
```env
# ─── Firebase ─────────────────────────────────
FIREBASE_API_KEY=your_key_here
FIREBASE_AUTH_DOMAIN=yourproject.firebaseapp.com
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_STORAGE_BUCKET=yourproject.appspot.com

# ─── AI Services ──────────────────────────────
OPENAI_API_KEY=your_openai_key
DIALOGFLOW_PROJECT_ID=your_dialogflow_id

# ─── Backend ──────────────────────────────────
API_BASE_URL=http://localhost:3000
```

**Step 4 — Launch 🚀**
```bash
npx expo start
# Press 'i' for iOS  |  'a' for Android  |  'w' for Web
```
